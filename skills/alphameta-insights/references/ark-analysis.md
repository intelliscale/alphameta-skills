# ARK Analysis / 创新分析

Disruptive innovation diagnostic — evaluates companies through ARK Invest's lens: TAM sizing, Wright's Law cost curves, S-curve adoption, and 3-scenario (Bull/Base/Bear) 5-year target pricing.

## Current Status

ARK-style analysis is a framework-only capability. AlphaMeta does not have:

- Dedicated TAM or Wright's Law modeling tools
- S-curve adoption analysis
- Scenario-based DCF modeling

**Approach**: Use WebSearch for industry research and TAM data. Use AlphaMeta's `financial-report`, `consensus`, and `operating` for company financials. Manual framework application is required — there is no automated ARK analysis pipeline.
